# Unified Mentor Projects

A collection of four comprehensive data analysis and visualization projects showcasing real-world data analytics skills including data cleaning, exploratory data analysis, statistical insights, and interactive dashboards.

## Projects Overview

### 1. **Climate NASA Project**
Analyzes NASA climate change data with social media sentiment analysis.
- **Dataset**: Climate change-related social media data
- **Key Findings**: Social media engagement patterns and climate sentiment trends
- **Outputs**: 
  - Jupyter Notebook with full analysis
  - Power BI Dashboard for visualization
  - Comprehensive report (PDF/DOCX)

### 2. **Cybersecurity Project**
Global network traffic analysis and attack detection.
- **Dataset**: CloudWatch traffic logs and network security data
- **Key Findings**: Traffic patterns, security anomalies, and network vulnerabilities
- **Outputs**:
  - Jupyter Notebook with network analysis
  - Power BI Dashboard for security metrics
  - Detailed security report (PDF/DOCX)

### 3. **IBM HR Analytics Project**
Employee attrition and turnover analysis for HR insights.
- **Dataset**: IBM HR employee data with attrition indicators
- **Key Findings**: Factors influencing employee turnover, retention patterns
- **Outputs**:
  - Jupyter Notebook with predictive analysis
  - Power BI Dashboard for HR metrics
  - Comprehensive insights report (PDF/DOCX)

### 4. **Uber Trip Analysis Project**
Trip demand analysis and performance metrics.
- **Dataset**: Uber trip data from January-February FOIL release
- **Key Findings**: Demand patterns, peak hours, trip distribution
- **Outputs**:
  - Jupyter Notebook with demand analysis
  - Power BI Dashboard for trip metrics
  - Trip demand analysis report (PDF/DOCX)

## Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook
- Power BI Desktop (for dashboard files)
- Required Python libraries: pandas, numpy, matplotlib, seaborn, scikit-learn

### Installation

```bash
# Clone or download the repository
cd Unified_Mentor_Projects

# Install Python dependencies
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

### Running the Analysis

1. Navigate to any project folder
2. Open the `.ipynb` file in Jupyter Notebook
3. Run cells sequentially to reproduce the analysis
4. View cleaned data in the respective `.csv` files
5. Open `.pbix` files in Power BI Desktop for interactive dashboards

## Files Description

Each project contains:
- **`.ipynb`** - Complete analysis with code, visualizations, and insights
- **`.pbix`** - Interactive Power BI dashboard
- **`*_cleaned_data.csv`** - Processed dataset ready for analysis
- **`*.csv/.xlsx`** - Raw source data
- **`.pdf/.docx`** - Executive reports with key findings and recommendations
